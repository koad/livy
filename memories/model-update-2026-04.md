# Model Update (2026-04-03)

**Framework:** koad:io is completely FREE to run. No paywall, no time limits, no feature locks.

**Sponsorship:** Optional. Peer ring membership at tiers — determines how many daemons you can connect to and what data flows.

## Tier Model

- **Free**: $0 — 5 peers, worker state only
- **Basic**: $50/mo — 20 peers, logs + metrics
- **Pro**: $200/mo — 100 peers, events
- **Enterprise**: Custom — unlimited, can sponsor others

## Key Concept Changes

**Portal:** kingofalldata.com is a **read-only namespace window** into live daemon state, not a SaaS product. Visit `yourname.kingofalldata.com` to see a live dashboard of your kingdom.

**Trust Ring:** Network of daemons connected via sponsorship bonds. Sponsorship = trust relationship + visibility rights. Free tier is the default.

**Why:** Peer connectivity (daemon-to-daemon network) is the product. Core framework must be free to build adoption. Sponsorship is how kingdoms coordinate at scale.
