# Getting Started with koad:io

koad:io is free, forever. No paywall, no hidden costs. You run the framework, build your workers, and coordinate your kingdom at zero cost.

## The Free Model

koad:io itself is free. What you pay for is **sponsorship** — membership in a trust ring where daemons connect and share data:

- **Free Tier**: $0 — Run one kingdom, sponsor up to 5 peers, see worker state
- **Basic Tier**: $50/mo — More peers, access to logs and metrics
- **Pro Tier**: $200/mo — Full data access, event streaming, larger rings
- **Enterprise**: Custom — Unlimited peers, can sponsor others, custom integrations

All tiers get the same core koad:io framework. The tier only affects how many kingdoms you can connect to and what data flows between them.

## Understanding koad:io

- **Your Kingdom**: A koad:io installation on your machine, running the daemon and workers
- **The Portal**: kingofalldata.com — a read-only window into live kingdom state
- **Trust Rings**: Networks of connected kingdoms linked by sponsorship
- **Entities**: Named identities (people, orgs, systems) that run kingdoms
