# Documentation Standards and Templates

This directory contains standards, templates, and style guides for creating consistent documentation across the koad:io ecosystem.

## Contents

- `style-guide.md` - Writing style and voice guidelines
- `templates/` - Reusable templates for common document types
- `markdown-conventions.md` - Markdown formatting standards
- `api-documentation-template.md` - Template for API documentation
- `readme-template.md` - Standard README.md template

## Using These Standards

All documentation in the koad:io ecosystem should follow these standards to ensure consistency and quality.
